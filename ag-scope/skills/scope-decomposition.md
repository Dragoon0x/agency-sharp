# Scope Decomposition
**Purpose:** Break client requests into specific, estimable deliverables — the translation from 'we need a new website' to a line-item scope.

## The Practice
Scoping is where money is made or lost. Every hour underestimated is margin that disappears. Every scope gap is a dispute waiting to happen.

## The Agency Context
Agency work has constraints that product teams don't — multiple clients, parallel projects, fixed budgets, external deadlines, and the constant tension between quality and speed. Scope Decomposition must account for these constraints to be practically useful.

## When to use
When this is the active constraint in client delivery or agency operations.
