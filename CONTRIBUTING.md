# Contributing

**agency-sharp** is opinionated by design. The bar is the same as every repo in this ecosystem — high.

## The bar

- Every skill encodes real agency operations judgment from people who've run real agencies and delivered real client work.
- Specific over generic. "Manage client expectations" is not a skill. "Scope creep happens in small, reasonable requests — 'can you adjust the mobile version?' — and the fix is not saying no but saying yes with a change order" is.
- No agency consultant speak. No frameworks that exist to sound impressive in pitch decks.
- If the skill could appear in a generic "how to run an agency" blog post without standing out, it's not ready.

## How to contribute

1. Fork the repo
2. Create a branch: `git checkout -b skill/your-skill-name`
3. Follow the SKILL.md format in the relevant domain
4. Open a PR with one line: what agency judgment does this skill encode?
