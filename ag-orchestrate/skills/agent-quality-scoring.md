# Agent Output Quality Scoring
**Purpose:** Evaluate and score agent outputs — the criteria and process for determining whether agent work meets the standard.

## The Practice
The layer nobody else has built. Individual agents are commodities. The orchestration system that coordinates them into a functioning agency — that's the competitive advantage.

## The Agency Context
Agency work has constraints that product teams don't — multiple clients, parallel projects, fixed budgets, external deadlines, and the constant tension between quality and speed. Agent Output Quality Scoring must account for these constraints to be practically useful.

## When to use
When this is the active constraint in client delivery or agency operations.
