# SEO Technical Implementation
**Purpose:** Implement SEO fundamentals — meta tags, structured data, sitemap, robots.txt, and the technical SEO that agency websites should ship with by default.

## The Practice
Engineering is where designs become real. Agency engineering has unique constraints — client codebases, tight timelines, and the need to hand off maintainable work.

## The Agency Context
Agency work has constraints that product teams don't — multiple clients, parallel projects, fixed budgets, external deadlines, and the constant tension between quality and speed. SEO Technical Implementation must account for these constraints to be practically useful.

## When to use
When this is the active constraint in client delivery or agency operations.
