# Code Review Process
**Purpose:** Run code reviews that catch quality, security, and maintainability issues — the specific review checklist for agency code.

## The Practice
Every quality failure in agency work happens at a handoff — where one phase ends and another begins, where one role passes work to another. Handoff design is defect prevention.

## The Agency Context
Agency work has constraints that product teams don't — multiple clients, parallel projects, fixed budgets, external deadlines, and the constant tension between quality and speed. Code Review Process must account for these constraints to be practically useful.

## When to use
When this is the active constraint in client delivery or agency operations.
