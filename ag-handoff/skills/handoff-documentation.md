# Handoff Documentation Templates
**Purpose:** Create standardized handoff documentation — templates that ensure nothing is missed at each transition point.

## The Practice
Every quality failure in agency work happens at a handoff — where one phase ends and another begins, where one role passes work to another. Handoff design is defect prevention.

## The Agency Context
Agency work has constraints that product teams don't — multiple clients, parallel projects, fixed budgets, external deadlines, and the constant tension between quality and speed. Handoff Documentation Templates must account for these constraints to be practically useful.

## When to use
When this is the active constraint in client delivery or agency operations.
